# Linux Command Practice

## Check Current Directory

```bash
pwd
```

## List Files

```bash
ls -lah
```

## Display Disk Usage

```bash
df -h
```

## Display Memory Usage

```bash
free -h
```

## Display CPU Information

```bash
lscpu
```

## Display Running Processes

```bash
ps -ef
```

## Search for Files

```bash
find . -name "*.sh"
```

## Search Text Inside Files

```bash
grep -rn "Linux" .
```

## Display Command History

```bash
history
```

## Display Network Interfaces

```bash
ip addr
```
