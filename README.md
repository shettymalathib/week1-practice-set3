## Bash Scripts

The `scripts/` directory contains the following Bash scripts:

### backup.sh

- Accepts a directory name from the user
- Creates a compressed `.tar.gz` backup of a directory entered by the user.
- Displays a success or error message


### system-health.sh

Displays:

- Hostname
- Kernel version
- CPU information
- Memory usage
- Disk usage

### file-management.sh

Demonstrates common Linux file operations, including:

- Creating directories
- Creating files
- Copying files
- Listing directory contents

##  How to Run the Scripts

### Make the scripts executable

```bash
chmod +x scripts/backup.sh
chmod +x scripts/system-health.sh
chmod +x scripts/file-management.sh
```

### Run the scripts

```bash
./scripts/backup.sh

./scripts/system-health.sh

./scripts/file-management.sh
```
